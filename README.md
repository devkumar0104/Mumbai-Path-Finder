# Mumbai-Path-Finder
A feature-rich web application (~2,000 lines) that replicates core mapping functionality for Mumbai's road network. Users can search locations, visualize routes, and get navigation guidance — all rendered in-browser without external map APIs.

## Features
- Search for locations across Mumbai
- Visualize routes between two points
- Get navigation guidance rendered directly in the browser
- No external map APIs used — everything runs in-browser

## How to Run
1. Clone the repository
2. Open `index.html` in any web browser
3. That's it — no installation or setup needed

## Tech Used
- HTML
- CSS
- JavaScript
- Geolocation API
